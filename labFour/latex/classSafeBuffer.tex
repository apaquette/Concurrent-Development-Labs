\doxysubsection{Detailed Description}
A Buffer Implementation. 

Uses C++11 features to implement a Buffer class that contains an array of events, as well as Semaphores to controll access to the events array. It contains \mbox{\hyperlink{classSafeBuffer_a34c4c8ca783ed7d80c14461258583e24}{put()}} and \mbox{\hyperlink{classSafeBuffer_aff5b0527f70cefef2d82d65c8854a100}{get()}} methods to add and remove events from the array.
